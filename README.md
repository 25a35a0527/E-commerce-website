# 🛒 E-Commerce Website

A responsive and modern E-Commerce website built using **HTML, CSS, and JavaScript**.  
This project provides a basic online shopping experience with cart functionality and clean UI design.

---

## 🚀 Features

- 🛍️ Product listing page
- 🛒 Add to cart functionality
- ➕ Increase / ➖ Decrease quantity
- 🧾 Automatic cart total calculation
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Clean and modern UI
- ⚡ Interactive user experience using JavaScript DOM

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
